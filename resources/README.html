📊 可视化监视面板 v1.16.0 增强版

独立窗口，提供流水线的实时可视化追踪。快捷键 Ctrl+Shift+M。

核心功能

  • 实时状态同步：主界面与监视面板双工执行，步骤状态跨会话保留
  • 行独立配置：每行可独立设置修复配置，覆盖全局配置
  • 产物预览：支持 MD/EPUB/PDF/Word 四种产物的系统默认应用预览
  • 修复预览配置闭环：从监视面板直接进入修复预览，调参后无缝写回独立配置
  • 步骤回滚：仅删除指定步骤产物，不影响下游文件
  • 中间文件清理：一键删除过程文件，只保留最终结果

📝 16项可配置修复

核心设计原则：不勾选 = 不修复。默认所有功能关闭，用户完全掌控。

📖 MD转EPUB — 精美排版电子书

基于 Pandoc + 自建 EPUB 打包器（EPUB 3.0 + NCX 兼容 EPUB 2.0）。

🎨 CSS 排版风格（外部文件自动发现）

风格特点适用
朴素样式白底黑字极简Kindle 墨水屏
简洁样式白色容器+彩色边框现代阅读器
暗色风格深色背景+浅色文字夜间阅读

✨ 核心特性

  • 10种主题颜色
  • YAML 标题作为 EPUB 内部标题（title/subtitle/author/date）
  • YAML 标题重命名输出文件
  • Mermaid 图表渲染（可选）
  • 图片自动处理（复制 + 去重 + 重名检测）

📄 EPUB转Word — 软回车修复 + 5种排版预设 v1.17.0 新增

基于 Calibre 转换，并内置强大的 XML 级 DOCX 后处理引擎。

🎨 5种排版预设

预设正文字体标题字体行距首行缩进说明
📚 学术论文宋体 / Times New Roman黑体1.5倍2字符适合论文/报告
📖 书籍排版宋体 / Times New Roman楷体1.5倍2字符适合电子书阅读
💼 商务报告微软雅黑 / Arial微软雅黑1.25倍无适合商业文档
💻 技术文档宋体 / Consolas黑体1.25倍无适合代码/API文档
📄 保留原样跳过所有样式修改，仅保留软回车修复

⚙️ 核心功能

  • ✨ 软回车智能修复：自动将 Word 中的软回车拆分为标准硬段落，彻底解决段落破碎问题
  • 📄 页面尺寸预设：扁平化单选支持 A4 / Letter / B5 / A5
  • 🧠 离焦状态记忆：切换模块或关闭窗口时自动保存当前选项，下次进入无缝恢复
